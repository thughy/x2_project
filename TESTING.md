# X² PROJECT - Testing Instructions

This document provides instructions for testing the X² PROJECT game.

## Requirements

- Godot 4.4 or higher

## Setup for Testing

1. Open the project in Godot
2. Run the project (F5 or press the Play button)
3. The Dev Tools Menu will appear

## Testing Process

### First Time Setup

1. From the Dev Tools Menu, select "Generate Placeholder Images"
2. In the placeholder generator scene, press SPACEBAR
3. Wait for the message "Placeholder generation complete!" in the console
4. Return to the Dev Tools Menu (via the editor or by restarting the game)

### Testing the Game

1. From the Dev Tools Menu, select "Start Game"
2. You should see the game scene with placeholder images
3. Test the following functionality:
   - Dialog advancement (SPACE or left mouse click)
   - Character emotion displays
   - Environment changes
   - Dialog choices (when presented)
   - Menu functionality (ESC key)

### Known Issues

- All images are placeholders - colored blocks with minimal text
- Some interactive features may be limited in this test version

## Reporting Issues

Please report any issues you encounter with:
1. A description of what happened
2. Steps to reproduce the issue
3. What you expected to happen
4. Screenshots if applicable

## Accessing Original Menu

If you need to access the original main menu directly, select "View Main Menu" from the Dev Tools Menu.
